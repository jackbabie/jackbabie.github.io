Git submodule là cái gì ?

30.12.2018 - - Reading time ~1 Minute

Nó đơn giản chỉ là quản lí nhiều repo trong cùng một thư mục, commit mà không ảnh hưởng tới phần khác ví dụ

content -> repo XYZ
-- blog -> repo A
-- website -> repo B
-- mail -> repo C

Bây giờ chỉ cần link repo tới thư mục tương ứng là có thể commit không ảnh hưởng gì đến repo chính

git submodule add -b master git@github.com:<USERNAME>/<USERNAME>.github.io.git blog
