Tạo site với Hugo nhanh lắm

30.12.2018 - - Reading time ~1 Minute

Cài client tool hugo

Mình xài mac nên cài khá là dễ: chỉ cần chạy

brew install hugo

thế là xong, nhưng mà cái này chạy hơi lâu một chút, chậm hơn so với gem jekyll, nói chung mình phải đợi cả mấy phút ấy.

Tạo site

Ở đây mình sẽ dùng tool vừa cài và tạo một site mới có cấu trúc kiểu static content

Chạy lệnh này

hugo new site jackbabie.github.io

Run site local

Mình chạy lệnh

hugo

thì nó sinh ra folder public/ để xài cho production

Thêm một bài viết

hugo new posts/hello-world.md

Cái này add themes thì cực kì đơn giản

  • cd themes
  • clone github repo của themes đó
  • chỉnh config file link themes tới tên folder themes

Deploy github

Chỉ việc push thư mục public/ lên github là được

Cái cảm giác làm được cái gì đó mới thấy vui vui lắm ….

git hugo
