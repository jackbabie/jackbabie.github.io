Error Deploy Git Ssh

31.12.2018 - - Reading time ~1 Minute

Vì trên máy sẽ cài đặt nhiều key ssh để deploy cũng như setup git, nên có trường hợp bị lẫn lộn và không deploy được.

Giải pháp: Check lại xem ssh đang dùng key nào và add lại key ssh cho đúng với các bước sau:

ssh -T git@github.com : Check xem nó đang dùng ssh key nào

ssh-add -l : show các ssh key

eval "$(ssh-agent -s)" : stop hết các pid đang chạy

ssh-add your_key : add lại key bạn dùng để deploy
